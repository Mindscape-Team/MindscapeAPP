## Overview
**Mindscape** is a comprehensive system designed to simulate a complete mental health experience by blending immersive technology with artificial intelligence. The project aims to recreate the atmosphere of a professional therapy session while providing emotionally intelligent interaction.

Mindscape combines a **3D virtual clinic**, an **empathetic AI chatbot**, and **multi-modal emotional analysis** to deliver a meaningful and human-centered mental health simulation.

---

## Key Features
- üè• **3D Virtual Therapy Clinic**  
  A fully interactive 3D environment that replicates the feel of a real professional therapy setting.

- ü§ñ **Empathetic AI Chatbot**  
  Guides users through personalized, supportive dialogue during sessions.

- üé≠ **Emotional Diagnosis System**  
  Detects and analyzes user emotions using:
  - Facial expression recognition  
  - Voice tone analysis  
  - Text-based sentiment analysis

## My Role
I was responsible for developing the **mobile application (with its backend) and 3D environment** using **Unity**, in collaboration with **Alaa Amer**.

### Responsibilities included:
- Designing and building the interactive **3D clinic environment** and user interface
- Integrating AI components such as:
  - Speech-to-Text (STT)
  - Text-to-Speech (TTS)
  - Chatbot responses
- Implementing **facial emotion recognition** and **voice emotion detection** systems
- Ensuring smooth interaction between the 3D environment and AI-driven features
